Welcome! This site was created to share a collection of CSS best practices. I learned a lot while working on it and I hope you learn a great deal from reading it!

I've divided the content up into two pages. The first covers how to write CSS efficiently. The second covers style guidelines.

Note that these are not hard rules for writing CSS. As long as your CSS validates and gives the desired appearances, you can write however you want. However these guidelines should help you be a better team player and code sites at lightning speed!
